Overview

Here, online interactive multimedia could be seen as an euphemism for networked video games, yet the topic may be a bit larger, including cases where some kind of online, persistent, multi-user virtual world has to be simulated, for example MMORPGs or any sorts of metaverse.

As for the topic of graphical 2D/3D rendering, it is specifically addressed in this section.

Networking Subsystem

Various architectures can be considered for networked applications, from a totally decentralised peer-to-peer one, to a strict client/server one, possibly based on an authoritative server (performing most of the world evaluation by itself, rather delegating some of this processing to clients).

Notably when an application is intensively interactive (ex: a real-time strategy game, as opposed as a turn-based one), compensating for the lag and jitter induced by the network is a difficult technical challenge.
